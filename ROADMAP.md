# Roadmap

## V1

- Dashboard une page.
- Grille compacte d'applications.
- Filtres de categories.
- Monogrammes automatiques.
- Recherche instantanee.
- Configuration YAML.
- URL interne/externe resolue automatiquement.
- Filtrage par visibilite `internal`, `external`, `both`.
- Image Docker generique.

## V2

- Interface d'administration.
- Creation et modification des applications.
- Import/export YAML.
- Favoris.
- Tags.
- Authentification.
- Utilisateurs.

## V3

- Widgets : horloge, date, meteo.
- Integrations : Uptime Kuma, Zabbix, Proxmox, Docker, Home Assistant.
- Flux RSS.

## V4

- Systeme de plugins.
- API publique.
- SDK.
- Marketplace.
- Notifications.
- Themes personnalisables.
